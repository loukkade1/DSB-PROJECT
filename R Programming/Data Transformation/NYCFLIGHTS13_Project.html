Question:

  1. Among all the flights, which airlines are considered on-time flights?
    1. Since this syntax only shows the carrier which it represents the airlines but it doesn‚Äôt show airline name. So we need to join two data frame between flights & airlines together to get the result.
      ontime_flights <- flights %>%
                  select(year, carrier, dep_delay, arr_delay)%>%
                  filter(dep_delay == 0 & arr_delay == 0)%>%
                  arrange(carrier)
    1. Join flights and airlines on carrier code:
      ontime_flights <- flights %>%
  left_join(airlines, by = "carrier")%>%
  select(year, carrier, name, dep_delay, arr_delay)%>%
  filter(dep_delay == 0 & arr_delay == 0)%>%
  arrange(carrier)
      • In this part, after you joining two data frame together, don‚Äôt forget to add name which it is ‚Äúan airline name‚Äù column in airlines data frame in select function after the left join. If you forget to put name in select function, it won‚Äôt show you the airline name column in ontime_flights data frame.
